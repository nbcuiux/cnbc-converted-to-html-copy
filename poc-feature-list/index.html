Breaking News

BREAKING NEWS

  • Single-page UI with Cards
    • Cards grouped by functionality
    • Foldable, rearrangeable cards
  • Contextual, role-based fields and options
    • E.g. Hot Seat Editor: Options to display homepage banner and send mobile notification
  • Assignment (non-mvp)
    • Select predetermined action, e.g. write article, insert image
    • Assign User or User Group
    • Set priority
    • Insert memo (optional)
    • View history of assignments
  • Simultaneous editing
    • Lock individual fields while in use
    • Simultaneous body editing per block (paragraph) level
    • Real-time update of the changed fields
  • Full rich text editor
    • Stylize
    • Hyperlink to stock ticker (Place, People, Organization, etc.) & external URL
    • Embed assets & external embed code
  • Autosaving
  • Social
    • Ability to share out to social media (Twitter)
    • Ability to preview the social post
    • Ability to override the og/twitter tags
  • Chat
    • With all the collaboratorswithin the content
    • @ mentioning to be integrated with Slack
  • Auto-population of fields:
    • Hyperlinked tickers to Primary Assets section
    • First image to Media card
    • Title to all the title-related fields
      • Title to SEO Title and Link Headline
      • Link Headline to Social Link Headline
      • SEO Title to Permalink Slug
  • Revision history
  • Preview
    • High fidelity preview

MEDIA

  • Media Library with images and videos
    • Other Media types, e.g. Chart, Audio, etc. can be added
  • Integration of 3rd-party sources e.g. Getty Images, AP Images, Reuters
  • One-click licensing
    • Only available for Photo Editors
  • Edit media items metadata
  • Focal point & rectangle
  • Image edit
    • Levels
    • Vignettes
    • Crop: if using services like Cloudinary
  • Filter out the photos that are last published 1 day, 2 days, 1 week, etc. ago
    • Only available for Photo Editors
    • Other users would only see the photos that are available for use, i.e. licensed, free-to-use, haven’t been used in the past 2 days

BREAKING NEWS FIELDS/ELEMENTS

  • Title/SEO Title/URL/Headline: Title of the article. SEO Title & URL & Headline default to match the Title, but are editable by the user.
  • Slug: Internal title
  • Permalink: URL of the article
  • Ability to send out red banner & push notification
  • Body: Text of the article
    • Needs initial body placeholder: This story is developing. Please check back for further updates. Click here for the latest on the markets.
  • Dateline: Date that gets displayed in the article.
  • Source
  • Text editor styling and embeds:
    • Subtitle
    • Hyperlink
      • Stock ticker
      • External links
    • Media (Image/Video)
    • Slideshow
    • External embeds, e.g. YouTube, Twitter, etc.
    • Pull Quote
  • Association to:
    • Franchise: Topic of the article / where the article appears on the front-end
      • Catastrophe
    • Primary Assets: articles appearing under that url – tags
    • Related Assets
  • Shorter Headline, Shortest Headline, Descriptions (SEO), Keywords
    • Different length for different usage, description and keywords for SEO.
    • Needs letter count – if the initial headline fits the character count of the other fields, automatically copy over?
    • Description is usually copied over from the lead
  • Role-specific options
    • Hot Seat: Display banner & send mobile notifications switches
    • Social Media Associate Producer: Social sharing of the story
  • Other general cards & elements
    • Settings & Configuration
      • Comments (option)
      • Family-friendly (option)
      • Dateline (option)
      • Byline (option)
      • Source (option)
      • Allow Syndication (option)
      • Premium Content (option)
      • Native (option)
      • Pre-roll on videos (option)
      • Template: Select Template
      • Skin: Select Skin
    • Team & Project
      • Team: Select Team, e.g. CNBC US Team
      • Project: Select Project
Dashboard

GLOBAL

  • Able to view latest published content within all of CMS
  • Performance Statistics (High priority)
    • Integration of 3rd party services: Omniture, Chartbeat, Domo (Referral tracking on social), Spredfast (Engagement – click, video play, reading the article, comment), Facebook Insights
    • Charts of the entire CNBC.coms performance, e.g. monthly visitors of the site
    • Charts of the articles’ performance, e.g. comments, views, reads, etc.
      • Any article that the user have edited
    • Charts of social media performance, e.g. Top 5 Facebook referrals
    • Import the preset charts in various analytics services
      • e.g. Filter by articles that have more than 50 comments, more than 500 views, etc.

PERSONALIZED CONTENT

  • Favorites – allows for shortcuts to pages
  • Able to view your latest activities
    • Published, Saved, and Assigned
  • Your Tasks (non-mvp)
    • Able to view the priority of the task
    • Each task is Immediately linked to the corresponding content
    • Tasks can be rearranged
    • Completed tasks are removed
    • Ability to instantly share to social media for Social Media team
  • Status Tracker
    • Track the status of the item in the list
      • Able to track the Publish, Save, and Assign actions made by different users
    • Item that you saved/published automatically gets added to the tracker
    • Able to add item to track from Content Library
    • Able to remove the item from the tracker
Global notification
  • Appears in the global bar of the CMS – can access anywhere within the system
  • Notified when there is an assignment (non-mvp)
    • Short summary of the assignment
  • Notified when the content that you are tracking is published
  • Option to go immediately to the notified content with a click
  • Slack integration for notification
